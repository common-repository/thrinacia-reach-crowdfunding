== Description ==

This plugin allows you to easily manage crowdfunding campaigns created on Thrinacia Reach at https://reach.thrinacia.com. You can easily locate and insert Thrinacia Reach crowdfunding campaign on any page of your WordPress site. The plugin will also automatically insert the widget code. For more information about Thrinacia Reach, please visit this page https://www.thrinacia.com/reach

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Enter your Thrinacia Reach email and password in the Thrinacia Reach Settings located on the menu to the left
4. Once enabled you should see on the bottom of page new options allowing you to list, search and add crowdfunding widgets to your content
5. The crowdfunding widget is displayed using shortcode when inserted into content, options can be passed in as desired through shortcode
